# ☕ Starbucks Global Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** built using Starbucks beverage data. It provides insights into beverage nutrition, caffeine content, calorie distribution, and category-wise analysis through dynamic visualizations and filters.

The dashboard is designed to demonstrate data cleaning, data modeling, DAX calculations, and dashboard design skills.

---

## 📊 Dashboard Features

- 📈 KPI Cards
  - Total Beverages
  - Average Calories
  - Average Caffeine
  - Average Sugar

- 📉 Interactive Visualizations
  - Calories Trend Analysis
  - Beverage Category Distribution
  - Top Beverage Comparison
  - Nutrition Breakdown
  - Category-wise Analysis

- 🎯 Interactive Slicers
  - Beverage Category
  - Beverage Type

- 🎨 Professional Dashboard Design
  - Starbucks-inspired theme
  - Custom icons and branding
  - Clean and responsive layout
